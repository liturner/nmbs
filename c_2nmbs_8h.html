Detailed Description

C API for certain aspects of the libnmbs.

These are primarily written to support in GLib like integrations, such as the Nautilus extension system. It is not intended to cover all aspect of the C++ API, rather it will be minimally expanded as needed. It is assumed that the primary users of the libnmbs will be using C++.
